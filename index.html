WHOIS Lookup

Domain & IP registration data via RDAP

Look up registration details for any domain name or IP address. Enter a domain to see its registrant, registrar, registration dates, nameservers, and status. For domains, the tool also resolves IP addresses and shows their geographic location.

All queries run directly in your browser using RDAP (the modern replacement for WHOIS), Cloudflare DNS for address resolution, and ipapi.co for geolocation. Nothing is logged or stored — no backend, no tracking.
